 Call Center Operations Dashboard | Power BI

  Project Overview
This project analyzes call center operational data using Power BI to track performance, identify call trends, and support data-driven decision-making. The dashboard focuses on call volume, call duration, sentiment analysis, and geographic distribution.

 Project Objectives
- Monitor overall call center performance
- Analyze call duration and agent efficiency
- Identify high-demand days, regions, and channels
- Understand customer sentiment distribution
